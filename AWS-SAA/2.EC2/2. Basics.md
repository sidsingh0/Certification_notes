EC2 is a Infrastructure as a service offering by Amazon.
It consists of the following:-
1. Renting Virtual Machines - EC2 Instances
2. Storages - 
	1. Network attached (EBS(Elastic Block Store) volumes)
	2. Hardware attached (EC2 Attached memory)
3. Distributing Load across machines (ELB)
4. Scale services using (ASG)

Configuration Options:-
- Operating System: Windows, Mac, Linux
- How much compute power
- How much ram
- How much storage
- Speed of the network card attached
- Public IP required
- Firewall rules and security groups for it
- Bootstrap script: Set of commands to be run at the start (ex. EC2 import user data)
		- It is done using *EC2 user data script*
		- Done to do things like
			- Update the os
			- Installing softwares
		- EC2 user data script runs with root users privilege(So be careful)

- We need Key Pair to access our created VM. We can do it using **SSH**.
	- Windows 10, Linux, Mac, OpenSSH -> `.pem` (command-> ssh path.pem `ec2-user`@ip)
	- Windows Putty -> `.ppk`
	- EC2 Instance connect is the easiest option to connect to ec2(no need to manage ssh keys, but still we need to enable port 22 for this to work)
- Public IP -> Public network (Internet)
- Private IP -> AWS's own private network IP
- IF WE STOP INSTANCE FOLLOWING HAPPENS:-
	- Public IP may change
	- Private IP stays the same
