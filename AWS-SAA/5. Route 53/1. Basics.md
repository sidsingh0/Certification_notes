- DNS is a service which translates human friendly hostnames into the machine IP address.
- Domain Registrar is a place where we register our domain names. (Ex. Amazon Route 53, GoDaddy, Google Domains etc.)
- DNS records: A, AAA, CNAME, NS
- Zone File: Contains all the DNS records.
