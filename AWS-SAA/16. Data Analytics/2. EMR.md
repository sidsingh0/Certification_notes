- Elastic Map Reduce
- Used for creating Hadoop Clusters - Used for analyzing Big Data
- Can be scaled to hundreds of EC2 instances.
- EMR comes bundled with Apache Spark, Fink, Presto, HBase
- Auto-scaling and integration with Spot Instances
- Use cases: Big Data processing, Machine Learning, Web Indexing

#### Node Types:
1. Master Node - Manage the cluster, co-ordinate, health (long running)
2. Core Node - Run tasks and store data (long running)
3. Task Node - Just run the tasks (spot)

#### Purchasing Options
1. On demand - Expensive, wont be terminated
2. Reserved: Savings (min 1 year), EMR uses this if available
3. Spot: Cheap, can be terminated, less reliable

We can have long running, as well as transient clusters

# Amazon Quicksight
- Used for creating dashboards for business intelligence
- Integration with machine learning
- Fast, serverless, embeddable.
- Pricing is per session
- Integration with 
	- RDS, Aurora, Athena, Redshift S3.
	- Salesforce, Jira
	- On premise (xlsx, csv, json, logs)
- *Uses SPICE, in memory engine that makes dashboard fast*
- *CLS(Column level security) - Different Users see different columns*
- We create users and groups INSIDE of Quicksight and not in IAM
- To create a dashboard, we first publish it. Then we can share it to users and groups.
- These users and groups can also see the underlying data.

# Glue Data Catalog
- Glue data catalog has a data crawler
- This crawler goes through the data sources (S3, RDS, DynamoDB, Onpremise) etc.
- Then it takes the metadata of these sources and writes it into the catalog
- This catalog is now accessible to Data analysis tools such as
	- Redshift Spectrum
	- Athena
	- EMR
- When required, these tools, using the metadata from catalog can locate the resource and access it.
