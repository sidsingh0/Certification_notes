- Perform bulk operations on existing s3 objects
	- Modify all metadata
	- Copy objects between s3 buckets
	- *Encrypt all the un-encrypted objects*
	- Modify Tags
	- Restore objects from glacier
	- Invoke lambda for processing on each object
- Job = list of object + action to perform + optional params
- We create a list of filtered objects using S3 Inventory and Athena query. Then we send this list of objects to **S3 Batch Operations** to produce processed outputs
- S3 Batch operations does the following:
	- Automatic retries
	- Report generation
	- Progress tracking
	- Completion notification
