- Tool that allows us to analyze and optimize our s3 storage.
- Discover Anomalies, Cost Optimization, Security practices
- Available on multiple levels, Organizations, Accounts, Regions, Buckets, Prefix
- Default dashboard enabled, create custom dashboards
- Available Metrics:
	1. Summary (totalBytes, totalObjects) = find fast growing buckets
	2. Security (MFA, Encryption, CrossRegionReplication, Versioning for buckets)
	3. Access Management (Bucket Owners)
	4. Performance (S3 Transfer Acceleration enabled buckets)
	5. Cost (Incomplete Multi Part, Non current version object bytes) = terminate incomplete multipart data, transition to archive non current versions
	6. Event Metrics (Notification enabled bucket count)
	7. Activiy Metrics (Get, put, post, list)
	8. Detailed status code metrics (403, 200 etc)
- Free vs paid
	- Free:
		- 28 metrics
		- 14 days data used for analytics
		- Automatically available for all
	- Paid:
		- More metrics, Advanced metrics
		- 15 months data
		- Prefix level data
		- Publish data directly to cloudwatch
