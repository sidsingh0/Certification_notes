## Methods to access AWS
- Management Console (UI): Password + MFA
- CLI: Access Keys (Generated via management consoles) 
- SDKs: API Keys

Do not share secret access keys with other users.
- Secret Access Keys ~ password
- Access key Id ~ username

## CLI Usage
- Configure the aws cli
- Open cmd and run `aws configure`
- Enter Access Id, Secret Access Key, Region
- To see a list of users: `aws iam list-users`
- To see list of ec2 instances: `aws ec2 describe-instances`

## Cloudshell
- It is a free CLI within the AWS management console
- Credentials/Region of the logged in user is used to connect to the cli.
- Files created inside cloudshell remain, even after logout.
- We can download and upload files to this cloudshell.
